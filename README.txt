RSS reader
Перед запуском программы перейдите в файл Settings/settings.xml
Настройте нужный сайт ленты, частоту обновления данных, а также при необходимости заполните данные
о прокси-сервере (адрес, локальный доступ, логин, пароль), если логина и пароля нет,то оставьте поля пустыми.
После настройки всех данных запустите приложение (через visual studio если нет iis, иначе можно настроить на самом iis с папкой publish2)
